# Clefs simplifiées sous Excalidraw


Je teste le tableau blanc collaboratif libre <a href="https://excalidraw.com">Excalidraw</a>, pour générer des clefs simplifiées illustrées par des photos sous licences libres  de Inaturalist.

Je conseille l'excellent logiciel libre <a href="https://pablocut.forge.apps.education.fr">Pablocut</a> pour le détourage automatique des photos.

Cela donne ceci :
<img src="https://raw.githubusercontent.com/daviddelon/Clefs_Plantes/refs/heads/main/geranium_petale_echancre_petite_fleur.png">

<a href="https://excalidraw.com/#json=yVKwdDTAf6vd_wtMRXK9S,QR4zzLcLNhk5dE4_HH0reQ">Exemple sous Excalidraw.

Attention ne pas utiliser l'exemple ci-dessus comme base de départ d'une nouvelle version, car ce n'est pas la dernière version. Pour travailler depuis la version en cours, vous pouvez par exemple télécharger un fichier se terminant par l'extension .excalidraw (dans Github clic sur un fichier puis icone téléchargement en haut à droite) et ensuite le charger dans  <a href="https://excalidraw.com">Excalidraw</a> (Menu ouvrir en haut à gauche).

Contributions bienvenues !
